# Utilities

Shared helper utilities used across the package, including:

- Dataclass helpers (`dataclass.py`).
- Logging setup (`logger.py`).
- Comparable and sorted dictionary helpers (`comparable.py`,
  `my_value_sorted_dict.py`).
- Miscellaneous small helpers (`small_tools.py`,
  `dict_of_numbered_dict_with_pointer_on_max.py`).
